Mysql安装以及mysql多实例配置

发表于 2017-10-05 | 分类于 数据库

安装前准备 下载：http://mirrors.sohu.com/mysql/MySQL-5.6/mysql-5.6.36.tar.gz 安装 Yum install ncurses-devel bison autoconf cmake gcc gcc-c++ 创建用户和组grouppadd mysql&& useradd mysql -s /sbin/nologin -M -g mysql 开始安装 解压：tar xf mysql-5.6.36.tar.gz 进入目录：cd mysql-5.6.36 编译参数 cmake . -DCMAKE_INSTALL_PREFIX=/application/mysql -DMYSQL_DATADIR=/application/mysql/data -DMYSQL_UNIX_ADDR=/application/mysql/tmp/mysql.sock -DDEFAULT_CHARSET=utf8 -DDEFAULT_COLLATION=utf8_general_ci -DWITH_EXTRA_CHARSETS=all -DWITH_INNOBASE_STORAGE_ENGINE=1 -DWITH_FEDERATED_STORAGE_ENGINE=1 -DWITH_BLACKHOLE_STORAGE_ENGINE=1 -DWITHOUT_EXAMPLE_STORAGE_ENGINE=1 -DWITH_ZLIB=bundled -DWITH_SSL=bundled -DENABLED_LOCAL_INFILE=1 -DWITH_EMBEDDED_SERVER=1 -DENABLE_DOWNLOADS=1 -DWITH_DEBUG=0 编译安装：make && make install 授权数据文件夹：chown -R mysql.mysql /application/mysql/data 初始化数据库：/application/mysql/scripts/mysql/mysql_install_db –user=mysql –basedir=/application/mysql –datadir=/application/mysql/data 拷贝配置文件以及启动文件：cp /application/mysql/my.cnf /etc/ &cp /application/mysql/support-files/mysql.server /etc/init.d/mysqld 启动mysql:/etc/init.d/mysqld start 添加环境变量：echo “export PATH=/application/mysql/bin:$PATH” >>/etc/profile 使环境变量生效：source /etc/profile 登录mysql：mysql -uroot

配置mysql多实例 简介：简单的说就是在一台机器开启多个不同的服务端口，运行多个mysql服务进程，这些服务进程通过不同的socket监听不同的服务端口来提供各自的服务。 创建多实例的相关目录：mkdir -p /data/{3306,3307}/data 授权：chown -R mysql.mysql /data 拷贝配置文件并修改 cp /etc/my.cnf /data/3306/ cp /etc/my.cnf /data/3307/ 3306文件夹下my.cnf修改如下参数 basedir = /application/mysql datadir = /data/3306/data port = 3306 server_id = 1 socket = /data/3306/mysql.sock pid-file = /data/3306/mysqld.pid 3307文件夹下my.cnf修改如下参数 basedir = /application/mysql datadir = /data/3307/data port = 3307 server_id = 2 socket = /data/3307/mysql.sock pid-file = /data/3307/mysqld.pid 分别初始化实例 /appplication/mysql/scripts/mysql_install_db –user=mysql –basedir=/application/mysql –datadir=/data/3306/data /appplication/mysql/scripts/mysql_install_db –user=mysql –basedir=/application/mysql –datadir=/data/3307/data 启动实例方法 mysqld_safe –defaults-file=/data/3306/my.cnf 2>&1 >/dev/null & mysqld_safe –defaults-file=/data/3307/my.cnf 2>&1 >/dev/null & 登录实例方法 mysql -uroot -S /data/3306/mysql.sock mysql -uroot -S /data/3307/mysql.sock 停止实例方法 mysqladmin -uroot -S /data/3306/mysql.sock shut down mysqladmin -uroot -S /data/3307/mysql.sock shut down
